# Security Policy

## Supported Versions

BookmarkBureau is a demonstration project. Only the latest version receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest (master) | :white_check_mark: |
| Older commits   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in BookmarkBureau, please report it by emailing **github@jelmerschreuder.nl**.

### What to include:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested fix (if you have one)

### Response timeline:

- **Initial response**: Within 72 hours
- **Status update**: Within 1 week with either a timeline for a fix or an explanation if the issue is not accepted
- **Resolution**: Security fixes will be prioritized and deployed as soon as possible

### Please do NOT:

- Open a public GitHub issue for security vulnerabilities
- Disclose the vulnerability publicly before a fix is available
